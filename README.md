# QuizMaster - Professional Online Quiz System

QuizMaster is a fully responsive and interactive online quiz platform developed using HTML5, CSS3, and Vanilla JavaScript. The application allows users to test their knowledge across multiple technical domains through a clean and modern user interface.

## Features

* Multiple quiz categories

  * Web Development
  * Python Programming
  * Database Design
  * Object-Oriented Programming
  * Artificial Intelligence
  * Cybersecurity

* Custom quiz settings

  * Adjustable timer
  * Selectable number of questions

* Interactive quiz experience

  * Previous/Next navigation
  * Real-time answer tracking
  * Dynamic progress bar
  * Countdown timer

* Advanced result system

  * Instant score calculation
  * Performance analytics
  * Answer review section
  * Correct vs incorrect answer visualization

* Responsive Design

  * Mobile-friendly UI
  * Tablet and desktop support
  * Modern gradient-based interface

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## Project Highlights

* Fully client-side application
* No external frameworks or libraries used
* Lightweight and optimized design
* Professional UI/UX implementation
* Responsive layout using Flexbox and CSS Grid
